Proyecto Final del Bootcamp de Machine Learning

En esta página se encuentra el último proyecto a realizar del Bootcamp Machine Learning ofrecido por la plataforma DIO Brasil, donde se presentan funcionalidades básicas respecto al procesamiento de lenguaje natural (PLN). Entre ellas, se incluye la capacidad de convertir texto en audio, reconocer lo que se habló y ejecutar comandos por voz para la búsqueda en el navegador.

Texto a Voz

Convierte texto escrito en audio de manera rápida y precisa.

Voz a Texto

Transcribe con precisión lo que dices en tiempo real.

Comandos por Voz

Controla funciones del navegador con tu voz de forma intuitiva.

Lista de Comandos
  • Abre YouTube: Abre la página principal de YouTube.
  • Abre el navegador: Abre el navegador web.
  • abre el mapa: Abre la página de Google Maps.
  • Abre Facebook: Abre la página de Facebook.
  • Abre Twitter: Abre la página de Twitter.
  • Abre WhatsApp Web: Abre WhatsApp Web en el navegador.
  • Abre el bloc de notas: Abre el Bloc de Notas en Windows.
  • Abre la calculadora: Abre la calculadora en Windows.
  • Apaga el sistema: Apaga el sistema (funciona en Windows).
  • Reinicia el sistema: Reinicia el sistema (funciona en Windows).
  • Abre el correo (Gmail): Abre la página de Gmail.
  • Abre mi carpeta de documentos: Abre la carpeta de documentos en el sistema.
